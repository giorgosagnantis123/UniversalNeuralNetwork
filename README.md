# UniversalNeuralNetwork

## Description
The goal of this research is to explore bioplausible gradient free learning and topologies that can offer state of the art performance while maintaining transparency and computational efficiency.

This is not a finished product but rather a proof of concept experimental implementation of my theoretical research on Artificial Intelligence.

Key characteristics:
   1) Use of discrete probabilistic pools and weight utility metrics
   2) Kernel like trick expansion which creates a random forest topology
   3) The flow of information on the linear layers is similar to that of a gated linear network
   4) Mixture of experts like approach and evolutionary weight pool strategies on the synaptic level
   5) Dampened algebraic inverse learning approach that leverages the network topology
      
This version is a stable first draft of the code. It is restricted on representational expressivity due to the lack of bias and non existing gated mechanism for weight picking. Those two additions and further small improvements are finished. The updated code will be available soon alongside proper test metrics.
